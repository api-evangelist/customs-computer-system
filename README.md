# Customs Computer System API (customs-computer-system)
APIs for interacting with customs computer systems for import/export declarations, tariff information, and trade compliance

**URL:** [Visit APIs.json URL](https://customs-api.example.com)

## Timestamps

- **Created:** 2024-01-01 
- **Modified:** 2024-01-01 

## APIs

### Customs Declaration API

**Human URL:** [https://customs-api.example.com/declarations](https://customs-api.example.com/declarations)


#### Tags:

 - Customs, Declarations, Import, Export, Trade

#### Properties

- [X-OpenAPI](https://api.customs.example.com/v1/declarations/openapi.json)
